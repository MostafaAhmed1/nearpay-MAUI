خطوات التكامل مع **NearPay SDK** لتطبيق موبايل (Android) لاستقبال مدفوعات العملاء عبر البطاقة أو NFC، موثقة حسب التوثيق الرسمي:

***

## 📋 المتطلبات الأساسية

| المتطلب | التفصيل |
| :-- | :-- |
| **نظام التشغيل** | Android 10+ (API 29) أو أعلى [^1][^2] |
| **NFC** | جهاز Android فعلي يدعم NFC ومفعّل [^1][^3] |
| **الموقع** | إذن الوصول للموقع (Location Permission) إلزامي [^2][^3] |
| **الاتصال** | Wi-Fi أو 4G فعال [^2] |
| **الأمان** | الجهاز غير م-rooted، غير محاكي (emulator)، bootloader مغلق [^2] |
| **حساب Sandbox** | يتطلبNearPay إنشاء حساب sandbox لك باستخدام إيميلك ورقم هاتك وـ **Android package name** [^1][^2] |


***

## 📝 خطوات التكامل بالتفصيل

### الخطوة 1: تسجيل طلب حساب Sandbox

أرسل إيميل إلى `support@nearpay.io` يحتوي على:

- إيميلك الشخصي
- رقم هاتفك
- **Android package name** لتطبيقك (مثلاً: `com.yourcompany.yourapp`) [^1][^2]

أرسل إيميل آخر إلى `onboarding@nearpay.io` بقائمة المستخدمين الذين يحتاجون الوصول لحساب Sandbox [^2].

***

### الخطوة 2: إضافة SDK إلى المشروع (Android Native/Kotlin)

في ملف `build.gradle`:

```gradle
dependencies {
    implementation("io.nearpay:nearpay-sdk-store:2.1.91")
}
```

> **ملاحظة لـ MAUI/Xamarin**: يوجد пакет `Xamarin.NearpayStoreSDK 2.1.88.1` متاح على NuGet لدعم Android MAUI/Xamarin [^4]

***

### الخطوة 3: تهيئة كائن NearPay

في أي مكان تحتاج فيه للـ SDK، أنشئ مثيل واحد:

```kotlin
val nearPay = NearPay.Builder()
    .context(this)
    .authenticationData(AuthenticationData.UserEnter)
    .environment(Environments.SANDBOX)  // أو Environments.PRODUCTION
    .locale(Locale.getDefault())
    .networkConfiguration(NetworkConfiguration.SIM_PREFERRED)
    .uiPosition(UIPosition.CENTER_BOTTOM)
    .paymentText(PaymentText("يرجى تمرير البطاقة", "please tap your card"))
    .loadingUi(true)
    .build()
```


***

### الخطوة 4: تنفيذ معاملة شراء (Purchase Transaction)

```kotlin
val amount: Long = 100  // يمثل 1.00 SAR (القيمة بالهللة)
val customerReferenceNumber = "9ace70b7-977d-4094-b7f4-4ecb17de6753"  // اختياري
val enableReceiptUi = true  // تفعيل واجهة الإيصالات
val enableReversal = true   // تفعيل زر الإلغاء
val finishTimeOut: Long = 10  // الثواني
val requestId = UUID.randomUUID()  // اختياري
val enableUiDismiss = true  // التحكم في إغلاق الواجهة

nearPay.purchase(
    amount, 
    customerReferenceNumber, 
    enableReceiptUi, 
    enableReversal, 
    finishTimeOut, 
    requestId, 
    enableUiDismiss, 
    object : PurchaseListener {
        override fun onPurchaseApproved(transactionData: TransactionData) {
            // الكود الخاص بك عند نجاح المعاملة
            // transactionData تحتوي تفاصيل المعاملة
        }
        
        override fun onPurchaseFailed(purchaseFailure: PurchaseFailure) {
            // الكود الخاص بك عند فشل المعاملة
        }
    }
)
```

## 🔄 الميزات المتاحة في SDK

| الميزة | الوصف |
| :-- | :-- |
| **Purchase** | شراء/استقبال دفعة [^1] |
| **Refund** | استرداد مبلغ لمعاملة سابقة [^5] |
| **Reverse** | إلغاء معاملة [^5] |
| **Reconcile** | التسوية اليومية [^2] |
| **Get Transaction** | جلب معاملة بـ UUID [^5] |
| **Get Transactions List** | جلب قائمة المعاملات [^5] |

## 📱 للتطبيقات عبر المنصات (Cross-Platform)

### Ionic/Cordova:

```bash
npm install @nearpaydev/nearpay-ionic-sdk --save
```

ثم للاستخدام:

```typescript
import { AuthenticationType, EmbededNearpay, Environments } from '@nearpaydev/nearpay-ionic-sdk';

const embededNearpay = new EmbededNearpay({
    authtype: AuthenticationType.email,
    authvalue: '<your-email>',
    environment: Environments.sandbox,
    locale: Locale.default
});

// الشراء
embededNearpay.purchase({
    amount: 1000,  // يعني 10.00 ريال
    transactionUUID: uuidv4(),
    enableReceiptUi: true,
    onPurchaseSuccess: receipts => console.log(receipts),
    onPurchaseFailed: err => console.log(err)
});
```


### iOS (Remote NearPay):

لـ iOS تستخدم **Nearpay Connect** عبر Bluetooth (ليس NFC مباشرة):

```swift
import NearpayConnectKit

@StateObject var conn: Connectivity = Connectivity.shared

conn.startPurchaseTransaction(amount: 300)
```


***

## ⚠️ ملاحظات هامة

1. **المدفوعات عبر NFC تعمل فقط على Android** - iOS يستخدم Remote NearPay عبر Bluetooth [^5]
2. **Developer Mode** يجب أن يكون مغلقًا [^2][^3]
3. **جهاز فعلي ضروري** - المحاكي لا يعمل مع SDK [^1][^2]
4. الإذن المطلوب في `AndroidManifest.xml`:

```xml
<uses-permission android:name="android.permission.ACCESS_FINE_LOCATION" />
<uses-feature android:name="android.hardware.nfc" android:required="true" />
```
